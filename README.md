# CivicFix – Public Problem Reporting System

Digital platform for citizens to report civic issues (roads, streetlights, garbage, sewage, water leaks) with photo + description, tracked by admins through Reported → In Progress → Resolved.

## Tech Stack
- Frontend: React, Tailwind CSS
- Backend: Node.js, Express, MongoDB
- Deployment: Vercel (frontend), Render (backend)
